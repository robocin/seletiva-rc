============
INTRODUCTION
============

The RoboCup Soccer Simulator Monitor (rcssmonitor) is used to view the
simulation as it takes place by connecting to the rcssserver or to
view the playback of a simulation by connecting to the
rcsslogplayer.

=================
USING THE MONITOR
=================

Run "rcssmonitor.exe" to start the monitor.

rcssmonitorr will look in the current working folder for the
configuration file:

`rcssmonitor.ini'

If this file does not exist it will be created and populated with
default values.
