# O que são os binários?

O binário é o formato que o código é enviado para as competições, tendo a restrição de não abrir o código e somente rodar.

# Como executar o binário
### 1. ITAndroids
```
cd itandroids/scripts/start
./start.sh
```
### 2. RobôCIn
```
cd /home/robocin/src
./start.sh
```

### 3. Titans

```
cd home/titans/src
./start.sh
```
